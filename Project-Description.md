# 📖 Project Description

## 📌 Abstract
NARLA is an AI-driven apartment co-pilot designed to simplify shared living by managing chores, groceries, bills, and household inventory. It combines task management, expense tracking, and intelligent insights to reduce conflicts and improve coordination among roommates. Built using a modern full-stack architecture, NARLA provides a centralized and user-friendly platform for efficient apartment management.

---

## 🧠 Overview
NARLA is a smart roommate management application built for students and shared households to organize and streamline daily apartment responsibilities. It provides a centralized platform where users can manage chores, track shared expenses, monitor groceries and inventory, and receive reminders.

The final version of NARLA focuses on usability, real-time updates, and a clean interface that allows users to quickly understand and manage the state of their household.

---

## 🎯 Problem Statement
Living with roommates often leads to challenges such as:
- Uneven chore distribution  
- Miscommunication about shared responsibilities  
- Difficulty tracking shared expenses  
- Lack of visibility into groceries and household inventory  
- Frequent conflicts over shared tasks  

These issues can create inefficiencies and tension in shared living environments.

---

## 💡 Final Solution
NARLA addresses these challenges by integrating multiple household management tools into one platform:
- A chore management system with assignments and tracking  
- A shared expense tracking system with flexible bill splitting  
- Grocery and inventory management  
- Notifications and reminders for tasks and payments  
- AI-driven insights to improve household coordination
- OCR-based receipt scanning for automatic item extraction and expense tracking

This unified system helps roommates stay organized, accountable, and informed in their shared living environment.

---

## ⚙️ Key Features
- 🧹 Chore assignment, tracking, and history  
- 🧾 Shared expense tracking and bill splitting  
- 🛒 Grocery list management with smart suggestions  
- 📦 Inventory tracking with low-stock alerts  
- 🔔 Notifications and reminders  
- 🤖 AI-generated household insights  
- 👥 Multi-user household system with roles
- 📸 Receipt upload with OCR-based item extraction for bills and groceries  

---

## 🏗️ System Architecture
NARLA follows a modern client-server architecture:

- **Frontend:** Next.js with React and Tailwind CSS for a responsive and user-friendly interface  
- **Backend & Database:** Convex.dev for real-time data handling, queries, and mutations  
- **Authentication:** Clerk for secure user authentication and session management  

The system supports real-time data synchronization across users, ensuring that household data remains consistent and synchronized.

---

## 🧰 Technologies Used

- **Backend:** Convex.dev
