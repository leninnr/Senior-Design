# 💰 Summary of Expenses

The NARLA project did not incur any direct monetary expenses during development. All tools, technologies, and resources used were either free, open-source, or available through student access.

The development of the application utilized technologies such as Next.js, React, Tailwind CSS, Convex, and Clerk, all of which were available under free or student-tier usage. Additionally, tools such as GitHub were used for version control and collaboration at no cost.

No hardware purchases or paid software licenses were required for this project. All development and testing were conducted using personal devices and existing resources.

Overall, the project was completed without any financial cost, demonstrating that a fully functional and scalable application can be built using accessible tools and platforms.
