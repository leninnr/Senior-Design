# 🎨 User Interface Specification

🔗 **Live Application:** https://narla.vercel.app/

---

## 📱 Overview
The NARLA user interface is designed to be clean, modern, and intuitive, enabling users to efficiently manage shared household responsibilities. The application uses a dark-themed gradient design with clear navigation and modular components for managing chores, bills, groceries, and inventory.

---

## 🧭 Navigation Structure
The application features a top navigation bar that allows users to switch between core modules:

- Dashboard  
- Chores  
- Bills  
- Inventory  
- Insights  
- Groceries  
- Household  

This structure ensures quick access to all features while maintaining a consistent user experience.

## 📊 Dashboard

![Dashboard Overview](OTHER/images/4%20dashboard.png)

- Displays household overview including:
  - Total expenses  
  - Monthly budget  
  - Household summary  
  - Recent activity  
  - AI insights  

- Quick action buttons:
  - Add Chore  
  - Add Bill  
  - Add Grocery  
  - Add Inventory  

**Supports User Story:**  
_As a roommate, I want to see a shared dashboard so that responsibilities and expenses are transparent._

---

## 🧹 Chores Management

![Chores Screen](OTHER/images/5%20chores.png)
![Add Chore Modal](OTHER/images/5%20add%20chore.png)

- View chores categorized as upcoming, overdue, and completed  
- Create and assign chores to roommates  
- Set frequency (daily, weekly, etc.)  
- Track workload distribution  

**Supports User Story:**  
_As a household, we want chores to be assigned and rotated fairly._

---

## 💳 Bills & Expense Tracking

![Bills Screen](OTHER/images/6%20bills.png)

- Track shared expenses and payment status  
- Visualize monthly spending trends  
- Upload receipts using OCR for automatic data extraction  

**Supports User Story:**  
_As the primary bill payer, I want to scan receipts and split costs automatically._

---

## 📦 Inventory Management

![Inventory Screen](OTHER/images/7%20inventory.png)

- Categorize items (pantry, fridge, cleaning, etc.)  
- Track quantities of items  
- Easily add, update, or remove items  

**Supports User Story:**  
_As a roommate, I want visibility into shared inventory._

---

## 🤖 AI Insights

![Insights Screen](OTHER/images/8%20insights.png)

- AI-generated recommendations based on household activity  
- Insights on chores, spending, and grocery usage  
- Helps improve coordination and efficiency  

**Supports User Story:**  
_As a busy student, I want AI-driven reminders and insights._

---

## 🛒 Grocery List

![Groceries Screen](https://raw.githubusercontent.com/leninnr/Senior-Design/main/homework_essays/OTHER/images/9%20groceries.png)

- Shared grocery list accessible by all household members  
- Items can be categorized (produce, snacks, cleaning supplies, etc.)  
- Status tracking for each item:
  - Needed  
  - Bought  
  - Skipped  

- Allows users to collaboratively add and update items in real time  
- Supports organized and efficient grocery planning for the household  

**Supports User Story:**  
_As a roommate, I want a shared grocery list so that everyone knows what needs to be bought._

---

## 🏡 Household Management

![Household Screen](OTHER/images/10%20household.png)

- Manage household members  
- Invite roommates using a unique code  
- Configure shared preferences such as budget and cleaning frequency  

---

## 🎯 Design Principles

- **Clarity:** Simple layouts with clear hierarchy  
- **Efficiency:** Quick actions reduce user effort  
- **Consistency:** Uniform design across all modules  
- **Real-Time Updates:** Changes reflect instantly across users  
- **Scalability:** Designed for future AI and hardware integrations  

---

## 🧾 Summary
The NARLA user interface provides a centralized, intuitive, and responsive platform for managing shared living. By combining usability with intelligent features, the interface ensures that users can easily coordinate tasks, track expenses, and maintain an organized household.
