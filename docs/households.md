# Households & Roommates

A **household** is the shared group for one apartment or shared living space.

---

## Creating a Household
1. Go to **Household**
2. Select **Create Household**
3. Enter:
   - Household name (ex: “12C at University Edge”)
   - Optional address/notes
4. Tap **Create**

![Household create placeholder](images/placeholder.png)

---

## Joining a Household
1. Get the join code or invite link from a roommate
2. Go to **Join Household**
3. Enter the code or open the link
4. Confirm

---

## Inviting Roommates
From the Household screen:
- Tap **Invite**
- Send via email or copy join code

---

## Roles (Admin vs Member)
Some households have roles:
- **Admin**: can manage members and settings
- **Member**: can use chores/bills/groceries normally

> If your team decides not to implement roles, remove this section.

---

## Removing a Roommate
Admins can remove members if someone moves out:
- Household → Members → Select member → Remove
