### 🔹 Part A: Individual Contribution

My main contribution to the NARLA project was working on the frontend, while also assisting with backend tasks when needed. I focused on designing and building the user interface to ensure the application was simple and intuitive for users.

Although frontend development was my primary responsibility, contributing to backend tasks helped me gain a better understanding of the full-stack architecture.

Throughout this project, I was able to build on the skills I identified in my initial assessment, particularly in frontend development and full-stack integration. I improved my abilities in:
- UI/UX design  
- Debugging  
- System integration  

One of my biggest accomplishments was helping take the project from an initial idea to a fully functional application that we can present.
