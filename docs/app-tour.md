# App Tour (What each screen does)

This section explains what you’ll see inside NARLA.

---

## Dashboard (Home)
The dashboard shows your household at a glance:
- Today’s chores
- Upcoming bills
- Grocery items running low
- Recent activity

---

## Household Tab
This is where you:
- See all roommates
- Add/remove members (if admin)
- Change household settings

---

## Chores Tab
This is where you:
- Add chores
- Assign chores
- Mark chores complete
- See history and reminders

---

## Bills Tab
This is where you:
- Add bills and due dates
- Split costs
- Track payment status

---

## Groceries / Inventory Tab
This is where you:
- Add shared items
- Track quantities
- See “low stock” alerts

---

## Notifications
You can receive reminders for:
- Upcoming chores
- Overdue bills
- Low inventory items

Notifications can be adjusted in Settings.
