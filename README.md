### Project Abstract
NARLA is a smart roommate app designed to simplify shared living. It keeps track of groceries, chores, and bills, while using AI to predict when essentials run out and to send timely reminders. By combining task management with fair expense and chore distribution, NARLA helps students avoid conflicts and stay organized in their day-to-day apartment life.
